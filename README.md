O programa huf faz a compressão dos dados, enquanto o unhuf faz a descompressão de dados comprimidos(.comp)

Comandos de compilação:

    g++ huf.cpp -o huf
    g++ unhuf.cpp -o unhuf

Comandos de execução:

    ./huf <nome_do_arquivo>
    ./unhuf <nome_do_arquivo>

    // Exemplo
    ./huf lorem_original.txt
    ./unhuf lorem_original.txt.comp
